# Employee Management System
Coding Challenge 6 - Using OOP to Model Roles and Hierarchies

## Description
An object-oriented employee management system built with JavaScript classes, demonstrating encapsulation, inheritance, and method overriding.

## How to Run
1. Open index.html in a browser
2. Right click → Inspect → Console tab
3. View employee list in console
